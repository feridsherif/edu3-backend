# Smart Contract Deployment Notes

1. Open the Solidity source at [educhain/contracts/CertificateRegistry.sol](../../../educhain/contracts/CertificateRegistry.sol).
2. Compile with Solidity 0.8.24 and enable optimizer runs.
3. Deploy to Polygon Amoy using Remix and a MetaMask wallet.
4. Store the deployed address in the BLOCKCHAIN_CONTRACT_ADDRESS environment variable.
5. Use the ABI from [src/blockchain/contracts/abi/CertificateRegistry.json](../contracts/abi/CertificateRegistry.json).
